# Videos 폴더

이 폴더에 AI 캐릭터의 MP4 영상 파일을 넣어주세요.

## 필요한 파일
- `daboki.mp4` - **다복이** 캐릭터의 말하는 영상 (필수)
- `ai-character.mp4` - 기타 AI 상담사가 말하는 영상 (선택)

## 영상 요구사항
- 파일 형식: MP4 (H.264 코덱 권장)
- 권장 해상도: 350x350px 이상
- 권장 파일 크기: 5MB 이하
- 권장 길이: 5-10초 (반복 재생됨)
- 음소거: 영상 자체 오디오는 자동 음소거됨 (TTS 음성만 재생)

## 파일 배치
```
public/
└── videos/
    ├── daboki.mp4        <- 다복이 캐릭터 영상 (CallPage에서 사용)
    └── ai-character.mp4  <- 기타 캐릭터 영상 (UserAppPage에서 사용)
```

## 사용 방법
1. **다복이** 선택 시: MainPage에서 "다복이"를 선택하고 "통화 시작" 클릭
2. CallPage로 이동하며 `daboki.mp4` 영상이 AI 음성과 동기화되어 재생됩니다
3. AI가 말할 때만 영상 재생, 말을 멈추면 영상 정지

자세한 사용 방법은 프로젝트 루트의 `VIDEO_SETUP.md` 파일을 참조하세요.
