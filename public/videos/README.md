# Videos 폴더

이 폴더에 AI 캐릭터의 MP4 영상 파일을 넣어주세요.

## 필요한 파일
- `ai-character.mp4` - AI 상담사가 말하는 영상

## 영상 요구사항
- 파일 형식: MP4 (H.264 코덱 권장)
- 권장 해상도: 400x400px 이상
- 권장 파일 크기: 5MB 이하
- 권장 길이: 5-10초 (반복 재생됨)

## 예제 파일 구조
```
public/
└── videos/
    └── ai-character.mp4  <- 여기에 영상 파일을 넣으세요
```

자세한 사용 방법은 프로젝트 루트의 `VIDEO_SETUP.md` 파일을 참조하세요.
